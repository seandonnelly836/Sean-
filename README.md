# Sean-# אפליקציית ניהול משימות (Task Manager) - פרויקט מסכם React

אפליקציית ניהול משימות מודרנית שנבנתה כפרויקט מסכם בקורס React. האפליקציה מאפשרת ניהול שוטף של משימות עם ממשק משתמש אינטואיטיבי ותמיכה מלאה בעברית.

## תכונות עיקריות
* **ניהול משימות:** הוספה, עריכת טקסט, סימון כבוצע ומחיקה.
* **סינון (Filtering):** אפשרות לצפות בכל המשימות, רק במשימות הפעילות או רק באלו שהושלמו.
* **שמירת נתונים:** שימוש ב-LocalStorage לשמירת המשימות בדפדפן גם לאחר רענון הדף.
* **מונה משימות:** תצוגה דינמית של מספר המשימות שנותרו לביצוע.
* **עיצוב רספונסיבי:** ממשק נקי התומך בכתיבה מימין לשמאל (RTL).

## רשימת קומפוננטות ואחריות
הפרויקט מחולק ל-5 קומפוננטות מרכזיות כדי לשמור על סדר, הפרדת רשויות ושימוש ב-Best Practices של React:

1. **App.jsx (קומפוננטת האב):**
   * מנהלת את ה-State המרכזי של האפליקציה (רשימת המשימות והפילטר הנבחר).
   * אחראית על הסנכרון מול ה-LocalStorage.
   * מכילה את הלוגיקה העסקית (הוספה, מחיקה, עריכה).

2. **TaskInput.jsx:**
   * קומפוננטה "נשלטת" (Controlled Component) לניהול שדה הקלט.
   * מבצעת ולידציה בסיסית (מניעת הזנת משימה ריקה).

3. **TaskFilter.jsx:**
   * מציגה את כפתורי הסינון.
   * אחראית לעדכן את קומפוננטת האב באיזה מצב תצוגה המשתמש נמצא.

4. **TaskList.jsx:**
   * מקבלת את רשימת המשימות המסוננת ומבצעת מיפוי (Map) להצגת הנתונים.
   * אחראית על העברת ה-Props לקומפוננטות הבן.

5. **TaskItem.jsx:**
   * מייצגת משימה בודדת.
   * מנהלת State פנימי למצב עריכה (Edit Mode).
   * מטפלת באירועי לחיצה לסימון ומחיקה.

## טכנולוגיות בשימוש
* **React 19** (Functional Components, Hooks: `useState`, `useEffect`).
* **Vite** ככלי בנייה מהיר.
* **CSS3** עם דגש על Flexbox ועיצוב RTL.

## הוראות הרצה
1. התקנת תלויות: `npm install`
2. הרצת הפרויקט במצב פיתוח: `npm run dev`
